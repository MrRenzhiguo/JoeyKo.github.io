## How To Get Started

**First, clone the repo**

```
git clone https://github.com/JoeyKo/JoeyKo.github.io
```

**Then**
```
npm install

npm start
```
**Listen at [http://localhost:8888/]([http://localhost:8888/])**
```
npm run build
```
